## Data Analysis Best Practices

1. **Understand data first**: Always examine data structure, types, and format before analysis
2. **Validate quality**: Check for missing values, outliers, duplicates, and inconsistencies
3. **Use appropriate methods**: Match statistical methods to data types and distributions
4. **Visualize clearly**: Generate charts with proper labels, legends, and titles
5. **Document methodology**: Record assumptions, parameters, and data transformations
6. **Present actionable insights**: Tie findings to decisions the user can make
